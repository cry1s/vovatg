# Форма, отправляющая данные в тг владельца

## Установка
1. Склонируй
2. Создай `.env` в корне проекта
3. В нём заполни TG_TOKEN_BOT
4. `npm start`
5. Отправь сообщение своему боту, скопируй его ответ
6. В `.env` заполни TG_CHAT_ID
7. Перезапусти

```python
TG_TOKEN_BOT = "<tgtoken>"
TG_CHAT_ID = "<chatid>"
```
